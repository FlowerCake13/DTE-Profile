AP Computer Science

AP Computer Science at Punahou starts with learning how to use the program Scratch, to learn basic scripting concepts through block coding. After learning Scratch, the course takes us through the CS50 curriculum, which is Harvardâ€™s introductory computer science course. Problem sets throughout the course teach C, Python, SQL, HTML, CSS, and JS. These problem sets are inspired by the arts, humanities, social sciences, and sciences. Through this class, you learn how to collaborate, problem-solve, and be a critical thinker, you also learn how to how to think algorithmically and solve problems efficiently.

Throughout the class, you work on many projects and problems, as well as a culminating project called the Create Task. The Create Task is a culminating project of the course that is turned in when taking the AP exam.
