Intro to App Development

Intro to App Development is an introductory course to the coding langauge Swift. Swift is used for mostly Apple apps, and some of these apps include VSCO, Bitmoji, AirBnB, WhatsApp, Instagram/Facebook, and 2048. This course goes through Apple's introductory course of SwiftUI. Using Swift playgrounds and the program xCode, you get to learn how to code in Swift, make apps for mobile phones and working on ways to make User Interfaces (UI).

By using Swift Playgrounds, a guided experience to learn Swift and basic program design with a fun twist, and creating your own apps, this course teaches you to work with others, share and recieve feedback, and be a critical thinker.
